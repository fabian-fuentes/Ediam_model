# Ediam vFrontiers

This repository contains scripts, data, and analyses for running the EDIAM model.

> **Note**: This copy of the repository is used for personal runs of the model and is not a production release.

## Folder overview

- `Main/` – entry-point scripts to execute policy experiments (`Main_P2.r`, `Main_P3.r`, `Main_P5.r`).
- `model/` – implementation of the International Green Tech Change Model.
- `RDM Inputs/` – experimental design CSV files and utilities (`create_experiment_function.r`, `harness_processing.r`).
- `RDM Outputs/` – example model outputs (`model.runs_fabian*.csv`).
- `RDM Harness/` – raw results generated by the harness runs.
- `analysis/` – R Markdown notebooks for scenario discovery and policy analysis, including
  `policy_p5_analysis.Rmd` which generates the policy P5 report.
- `auto/` – GitHub Pages output containing the generated reports:
  [`index.html`](auto/index.html) and [`policy_p5_analysis.html`](auto/policy_p5_analysis.html).
- `docs/` – rendered HTML reports.

The repository previously referenced scripts such as `ClimateCalibration.r` and `Ediam_vFrontiers.r`. These have been refactored into the files under `Main/` and `analysis/`.
