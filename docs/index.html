Acerca del Modelo EDIAM

Este repositorio contiene todos los scripts de soporte y conjuntos de datos necesarios para replicar el análisis presentado en Edmundo Molina-Perez et al, (2020) sobre inteligencia computacional para estudiar desafíos de sostenibilidad.

Scripts Principales

🌡️ ClimateCalibration.r

Método para determinar coeficientes climáticos del modelo usando datos raw de GCMs.

🏗️ Ediam_vFrontiers.r

Estructura matemática del modelo EDIAM y procesamiento del diseño experimental.

⚙️ Main_vFrontiers.r

Rutinas para diseño experimental, respuesta óptima de políticas y post-procesamiento.

🔍 sdprim_vFrontiers.r

Análisis con algoritmo PRIM (Patient Rule Induction Method).
